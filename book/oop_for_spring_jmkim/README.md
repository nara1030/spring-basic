## 구성

* [1장_사람을 사랑한 기술](ch_1.md)
* 2장_자바와 절차적/구조적 프로그래밍
* 3장_자바와 객체 지향
* 4장_자바가 확장한 객체 지향
* 5장_객체 지향 설계 5원칙 - SOLID
* 6장_스프링이 사랑한 디자인 패턴
